# Batty_et_al_Nature_Comms_2026
This repository contains the Ipython notebooks used to analyse the data generated in the Batty et al, 2026 publication 'Disruption of the SAGA CORE triggers collateral degradation of KAT2A' from the Seruggia lab, published in Nature Communications.

- The 'image_analysis_pipeline' folder contains an example ipython notebook for the image analysis pipeline used in the publication. Additional notebooks for filtering the data and downstream plotting can also be found here. These notebooks were used to analyse data in the following figures: 1e, 2b, 2d, 2j, 4b, 4e, 5g, 6b, S3i, S5b, S5e, S5h, S6b, S6f, S9g, S9i.
- The 'TMT proteomics' folder contains notebooks to recreate figures used for TMT-expression data, found in the following figures: 4e, 8a-e. The notebooks allow plotting of data as heatmaps, volcano plots, and calculation and plotting of the number of detected peptides in the dataset.
- The 'volcano_plots_crispr_screen' contains a notebook to recreate the volcano plot of the pooled CRISPR screen used to find effectors regulating KAT2A protein stability in Fig. 5e.
